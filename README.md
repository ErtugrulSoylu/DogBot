# DogBot
A discord music bot that can be used in more than one discord guild.

Commands:
  -queue
  -clear
  -clean
  -play
  -stop
  -resume
  -pause
  -join
  -leave
  -skip

What can DogBot do:
  -Play a song from url
  -Play a youtube playlist
  -Can play in different guilds
  -Can get better quality due to using local server
